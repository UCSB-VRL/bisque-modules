Volumetric Segmentation of Brain CT Scans

About:

This module is for segmenting a CT scan of a brain into 3 parts: lateral ventricles, cerebral matter, and subarachnoid space using a 3D UNet. A Support Vector Machine is then used to determine if the subject has possible Normal Pressure Hydrocephalus (NPH).

Paper: Automated Segmentation of CT Scans for Normal Pressure Hydrocephalus.

Input

The input should be a single image or a folder of images in nifti format (ending in .nii.gz).

Outputs

  • Info: CSV file of volumetric measurements and NPH prediction.
  • Segmentation Mask:A segmentation mask in nifti format. If folder of images is used, a folder of segmentation masks are returned.
